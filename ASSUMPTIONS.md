# Modeling Assumptions and Interpretation Notes

## 文件目的 / Purpose

本文件記錄本專案視覺化與計算模型背後的前提假設。它的目的不是證明地外文明存在，也不是估計宇宙中文明的絕對數量，而是界定本專案的推論範圍。

核心問題是：

> 如果生命世界或技術文明廣泛分布，地球式射電偵測方法會優先看見哪一類樣本？

因此，本專案討論的是觀測者選擇偏誤，而不是生命稀有性的直接判斷。

## 1. 生命普遍性假設 / Life Prevalence

本模型可容納相對寬鬆的生命普遍性假設：

- 生命可能在適宜行星上相當常見。
- 簡單生命可能廣泛分布。
- 技術文明可能在多個地點出現，而非地球獨有。

然而，模型明確區分「存在」與「可偵測性」。一個文明即使存在並發展出射電技術，也可能因為訊號無法跨越時間與能量門檻，而對地球保持不可見。

### 解讀

若生命或文明是普遍的，底層真實分布可能仍然廣泛且普通；但可觀測分布可能被少數高功率、近距離，或在時間上很早具備射電能力的來源主導。這正是本專案想呈現的選擇偏誤。

## 2. 文明發展速度假設 / Development Speed

本模型不假設所有文明都與地球以相同速度發展。相反地，介面提供可調整的 `lead_time` 參數，代表一個相對普通文明可能比地球提前多少年進入射電技術階段。

常見解讀區間：

| `lead_time` | 解讀 |
|---:|---|
| `25-100` 年 | 與地球接近的文明，射電技術只略微領先 |
| `100-1000` 年 | 工程能力明顯領先，但不必然具備恆星級能源控制 |
| `> 1000` 年 | 技術軌跡可能已難以與地球射電發展史直接類比 |

### 解讀

較短的領先時間會使許多普通文明在物理上存在、但在觀測上不可見。較長的領先時間會增加時間上可行的目標數量，但距離造成的能量衰減仍可能排除大量遠距系統。

## 3. 射電發射假設 / Radio Emission

本模型採用簡化的射電通訊假設：

- 發射功率被視為有效全向功率，或經波束修正後的等效功率。
- 訊號衰減依照距離平方反比關係。
- 若有效功率跨越所選門檻，即視為可被偵測。
- 模型不細緻模擬星際閃爍、吸收、頻寬、編碼、都卜勒漂移、占空比或搜尋策略完整性。

### 解讀

畫面中的門檻應被理解為概念性的物理過濾器，而非特定觀測站的完整性能預報。其目的在於展示距離與能量限制如何塑造可觀測樣本。

## 4. 訊號時間假設 / Signal Timing

對於距離 `d` 光年的目標，若地球於年份 `T_observed` 接收到訊號，模型將發射年份表示為：

```text
T_emit = T_observed - d
```

目前專案採用：

```text
T_observed = 2026
```

因此，距離越遠的目標，訊號必須越早被發射。這會形成一個技術領先下限：遠距發射源必須在地球接收年代之前很久就已具備相應能力。

## 5. 能量門檻假設 / Energy Threshold

模型使用簡化平方反比關係估算所需等效全向發射功率。若接收端最低可偵測通量門檻為 `F_min`，則：

```text
P_required = 4πd²F_min
```

本專案將 `4πF_min` 合併為比例常數 `K`，得到：

```text
P_required = K * d^2
```

目前資料產生流程採用：

```text
P_required = 1.12e11 * distance_ly^2
```

這裡的 `1.12e11` 是以 Five-hundred-meter Aperture Spherical radio Telescope（FAST）作為高靈敏度地球接收端參考時，折算出的等效門檻常數。它的功能是提供一個接近現代高靈敏度射電觀測能力的 baseline，使模型能比較不同距離目標所需的等效全向發射功率。

其中 `distance_ly` 是由 NASA Exoplanet Archive 中的 `sy_dist` 欄位換算而來：

```text
distance_ly = sy_dist * 3.26156
```

### 解讀

這個功率模型的重點不是提供精確工程設計，而是展示即使在線性距離增加時，所需功率也會以平方尺度快速上升。遠距目標若仍可被看見，通常意味著發射端具有極高有效功率、波束增益，或其他強化可偵測性的條件。

在射電天文中，接收靈敏度並不是單一固定的「最小可接收功率」。實際偵測能力通常取決於 flux density、SEFD、系統溫度、頻寬、積分時間、訊噪比門檻、偏振數、訊號漂移率與搜尋管線。本模型為了維持視覺化可操作性，將這些因素折算為單一 `K` 值。因此，`P_required` 應理解為 FAST baseline 下的概念性 EIRP 門檻，而非 FAST 在任一特定觀測模式下的完整靈敏度預報。

## 6. FAST 接收端基準 / FAST Receiver Baseline

選用 FAST 作為接收端基準的理由是：FAST 是目前極具代表性的高靈敏度單口徑射電望遠鏡，適合作為「地球現有高階射電偵測能力」的參考點。

本模型中的 FAST baseline 代表：

- 以現代高靈敏度地球射電望遠鏡作為接收端。
- 將接收端門檻折算為等效通量門檻。
- 再由平方反比關係轉換為目標端所需的等效全向發射功率。

本模型中的 FAST baseline 不代表：

- FAST 官方所有接收機與頻段的完整性能表。
- 特定 SETI 觀測計畫的完整 radiometer equation。
- 對任何單一系外行星目標的正式可偵測性預報。

展示本專案時，建議將此參數描述為「以 FAST 作為代表性高靈敏度接收端所折算出的概念性偵測門檻」，而不是簡化成「FAST 可以接收到的最小能量」。

## 7. 可觀測分布假設 / Observed Distribution

本專案區分兩種分布：

- 底層分布：假設中真實存在的生命世界或技術文明分布。
- 可觀測分布：通過射電偵測方法之時間與能量篩選後留下的子集合。

即使底層分布廣泛且普通，可觀測分布仍可能偏向：

- 距離較近的系統。
- 發射功率異常高的來源。
- 在時間上大幅領先地球的來源。
- 具備第一型、第二型或更高有效能源尺度的文明。
- 因定向發射幾何而提高有效接收通量的案例。

### 解讀

在此框架下，未偵測到普通文明不必然表示普通文明稀少。它也可能表示目前的偵測方法優先讓極端案例進入可觀測樣本。

## 8. 本模型不主張的事項 / Non-Claims

本專案不主張：

- 地外生命必然普遍。
- 技術文明必然大量存在。
- 任一特定系外行星必然存在生命或智慧。
- 所有文明都使用射電通訊。
- 所有射電通訊都是全向發射。
- 地球必然能偵測到先進文明。
- 目前的簡化功率模型等同完整射電天文儀器模型。
- FAST baseline 等同 FAST 在所有觀測條件下的固定最低可接收功率。

## 10. 主要解讀主張 / Main Claim

本專案的主要主張是條件式的：

> 如果生命或文明廣泛分布，而地球又透過受光速延遲與平方反比衰減限制的射電方法搜尋，則可觀測樣本仍可能高度不代表底層真實分布。

在這種情況下，地球看似位於「通訊技術窪地」，並不一定是因為普通文明不存在，而可能是因為普通文明在進入可觀測樣本之前，就已被物理條件過濾掉。
